# 浏览器缓存清理指南

## 问题症状
- 点击"保存并退出"按钮时，控制台显示："创建空白模板失败"
- 保存功能不工作
- 不会返回列表

## 根本原因
浏览器缓存了旧版本的 JavaScript 代码（app.js）

## 解决方案

### 方案1：强制刷新（推荐）
1. 在浏览器中打开 https://review-system.pages.dev
2. 使用键盘快捷键强制刷新：
   - **Windows/Linux**: `Ctrl + F5` 或 `Ctrl + Shift + R`
   - **Mac**: `Cmd + Shift + R`
3. 再次尝试编辑和保存复盘

### 方案2：清除缓存
1. 打开浏览器开发者工具（F12）
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"（Chrome）或"清除缓存"（Firefox）
4. 关闭并重新打开页面

### 方案3：无痕/隐私浏览模式
1. 打开无痕窗口（Chrome: Ctrl+Shift+N，Firefox: Ctrl+Shift+P）
2. 访问 https://review-system.pages.dev
3. 测试编辑和保存功能

### 方案4：清除站点数据（最彻底）
#### Chrome:
1. 按 F12 打开开发者工具
2. 右键点击地址栏左侧的锁图标
3. 选择"网站设置"
4. 点击"清除数据"
5. 刷新页面

#### Firefox:
1. 地址栏输入：`about:preferences#privacy`
2. 滚动到"Cookies 和网站数据"
3. 点击"管理数据"
4. 搜索 "review-system.pages.dev"
5. 删除所有相关数据
6. 刷新页面

## 验证修复
完成上述步骤后，验证以下功能：

1. ✅ 按钮文本显示为"保存并退出"（不是"保存"）
2. ✅ 点击按钮后，控制台不再显示"创建空白模板失败"
3. ✅ 点击按钮后，显示"更新成功"通知
4. ✅ 0.8秒后自动返回"我的复盘"列表
5. ✅ 数据已成功保存
