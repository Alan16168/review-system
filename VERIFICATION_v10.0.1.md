## 🐛 常见问题

### Q1: 我看到的还是"智能體配置"？

**A**: 请清除浏览器缓存：
- Chrome: `Ctrl+Shift+R` (Windows) 或 `Cmd+Shift+R` (Mac)
- 或者使用无痕模式访问

### Q2: 切换语言后没有变化？

**A**: 
1. 刷新页面（F5）
2. 检查右上角语言选择器是否已切换
3. 清除浏览器缓存后重试

### Q3: 其他页面的繁体中文正确吗？

**A**: 本次修复**仅针对 Admin Panel 菜单**，其他页面的翻译未修改。
