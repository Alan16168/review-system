### 2. 功能同步验证 (V8.5.0-V8.6.1) ✅

**V8.5.0: 多层AI服务回退**
- [ ] 进入"名著分析"
- [ ] 创建新分析（文本或视频）
- [ ] 如果Gemini失败，检查是否自动尝试OpenAI
- [ ] 查看分析结果页面显示使用的服务源（gemini/openai/claude/genspark）
- [ ] 错误提示显示详细的错误列表

**V8.5.1: YouTube字幕提取**
- [ ] 进入"名著分析"
- [ ] 选择"视频"输入类型
- [ ] 输入YouTube链接：`https://www.youtube.com/watch?v=dQw4w9WgXcQ`
- [ ] 验证能成功获取字幕（不会返回空内容）

**V8.6.0: 字幕预览功能**
- [ ] 输入YouTube视频链接
- [ ] 点击"开始分析"
- [ ] **应该显示字幕预览页面**（而不是直接分析）
- [ ] 预览页显示：
  - 视频标题、频道、时长
  - 字幕语言和字数统计
  - 字幕内容（最多5000字）
- [ ] 有"返回修改"和"确认并分析"两个按钮
- [ ] 点击"确认并分析"才开始AI分析

**V8.6.1: 复盘列表分类**
- [ ] 进入"审查"页面（底部导航第二个）
- [ ] **验证只显示普通复盘**（不含名著复盘和文档复盘）
- [ ] 进入"名著分析"页面
- [ ] **验证只显示名著类型的复盘**
