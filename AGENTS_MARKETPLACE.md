## 技术实现

### 后端 (Hono + TypeScript)
- **路由**: `/api/agents` - 智能体列表API
- **文件**: `src/routes/agents.ts`
- **功能**: 
  - `GET /api/agents` - 获取所有智能体
  - `GET /api/agents/:id` - 获取单个智能体详情

### 前端 (Vanilla JavaScript)
- **文件**: `public/static/agents.js`
- **对象**: `AgentsPage`
- **功能**:
  - 智能体列表展示
  - 分类筛选
  - 详情弹窗
  - 使用跳转
  - 视图切换

### UI设计
- **框架**: Tailwind CSS
- **图标**: Font Awesome
- **风格**: 现代卡片设计
- **交互**: 
  - 悬停效果
  - 平滑过渡
  - 响应式布局

## 使用流程

1. **访问商城**
   - 点击主菜单"商城" > "智能体商城"

2. **浏览智能体**
   - 查看所有可用智能体
   - 按分类筛选
   - 查看功能特性和评分

3. **查看详情**
   - 点击"详情"按钮
   - 查看完整描述和功能列表
   - 了解使用统计

4. **使用智能体**
   - 点击"使用"按钮
   - AI写作会自动跳转到AI写作页面
   - 其他智能体显示"正在启动"提示

5. **我的智能体**
   - 点击"我的智能体"查看已使用的智能体
   - 当前为空状态，显示"noAgents"
   - 可点击"前往商城"按钮快速跳转

## 开发说明

### 添加新智能体
编辑 `src/routes/agents.ts`，在agents数组中添加：

```typescript
{
  id: 5,
  name: '新智能体',
  icon: '🔥',
  description: '描述文字',
  category: '分类',
  features: ['特性1', '特性2'],
  status: 'active', // 或 'coming_soon'
  usageCount: 0,
  rating: 0,
  created_at: new Date().toISOString()
}
```

### 集成新功能
在 `public/static/agents.js` 的 `useAgent` 函数中添加跳转逻辑：

```javascript
if (agent.name === '新智能体') {
  window.location.hash = '#new-page';
}
```
