# 📖 用户指南：编辑"是否允许多个复盘答案"设置

## 🎯 功能说明

现在您可以在编辑复盘时，修改"是否允许多个复盘答案"的设置！

- ✅ **设置为"是"**：可以创建多个答案组
- ✅ **设置为"否"**：只有一个答案组

**注意**：只有复盘的创建者才能修改此设置。

---

## 📋 使用步骤

### 1. 打开复盘编辑页面

1. 登录系统：https://review-system.pages.dev
2. 进入"我的复盘"
3. 点击要编辑的复盘
4. 点击"编辑"按钮

### 2. 找到设置

在编辑页面的"复盘表头"部分，找到"是否允许多个复盘答案"设置：

```
┌─────────────────────────────────────────┐
│ 是否允许多个复盘答案 *                  │
│                                         │
│ ○ 是 (可创建多个答案组)                 │
│ ○ 否 (仅一个答案组)                     │
│                                         │
│ ⚠️ 修改此设置后将刷新页面              │
└─────────────────────────────────────────┘
```

### 3. 修改设置

**选择"是"**：
- 点击"是"单选按钮
- 系统自动保存
- 页面自动刷新
- 显示"答案组管理"功能
- 可以创建多个答案组

**选择"否"**：
- 点击"否"单选按钮
- 系统自动保存
- 页面自动刷新
- 隐藏"答案组管理"功能
- 只保留一个答案组

## 💡 常见问题

### Q1: 我看不到"是否允许多个复盘答案"设置

**A**: 请确认：
- ✅ 您是该复盘的创建者
- ✅ 您已登录系统
- ✅ 您在编辑页面（不是查看页面）

### Q2: 设置显示为灰色，无法点击

**A**: 这表示：
- 您不是该复盘的创建者
- 只有创建者可以修改此设置
- 您只能查看当前设置值

### Q3: 修改后答案组管理功能没有显示/隐藏

**A**: 请尝试：
1. 刷新页面（按F5或Cmd+R）
2. 清除浏览器缓存
3. 强制刷新（Ctrl+Shift+R 或 Cmd+Shift+R）
4. 查看浏览器Console（F12）是否有错误

### Q4: 从"是"改为"否"后，之前的答案组怎么办？

**A**: 
- 已有的答案组仍然存在
- 数据不会丢失
- 只是界面上隐藏了创建新答案组的功能
- 您仍可以查看和编辑现有答案组

### Q5: 修改设置后页面会自动刷新吗？

**A**: 
- ✅ 是的，修改后会自动刷新
- 刷新后会保持在编辑页面
- 答案组数据会重新加载
- 您可以继续编辑复盘

## 📊 使用场景建议

### 场景1：个人复盘（建议：否）

**适合**：
- 每次只需填写一份答案
- 简单的个人日志
- 不需要多次填写

**好处**：
- 界面更简洁
- 操作更直接
- 专注于内容

### 场景2：团队协作（建议：是）

**适合**：
- 多人需要提交答案
- 需要对比不同时间点的答案
- 需要保留历史版本

**好处**：
- 每个人可以有自己的答案组
- 可以创建多个时间点的记录
- 便于对比和分析

### 场景3：定期回顾（建议：是）

**适合**：
- 周报、月报等定期复盘
- 需要查看历史变化
- 需要追踪进度

**好处**：
- 每次填写创建新答案组
- 保留完整历史记录
- 便于趋势分析

## ⚠️ 重要提示

### 1. 只有创建者可以修改

- 团队成员无法修改此设置
- 如需修改，请联系复盘创建者
- 或创建者可以将复盘转让给您

### 2. 修改会自动保存

- 选择新值后立即保存
- 无需点击"保存"按钮
- 页面会自动刷新

### 3. 数据不会丢失

- 改变设置不会删除数据
- 已有答案组仍然存在
- 只是界面显示有变化

### 4. 建议在编辑前设置

- 最好在开始填写前设置
- 避免频繁修改
- 保持团队使用习惯一致
