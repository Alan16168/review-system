## 📚 Famous Book Review 使用流程

### Step 1: 填写表单
- 选择输入类型：
  - ⭕ **视频链接**（YouTube、Bilibili 等）
  - ⭕ **著作名称**（推荐使用原著名称）
- 输入**字数要求**：500-10,000 字
- 选择**应用场景**：
  - 职场
  - 创业
  - 个人成长
  - 个人财务规划
- 选择**输出语言**：
  - 英文、法文、西班牙文
  - 简体中文、繁体中文
  - 日文

### Step 2: 生成并编辑 Prompt
- 点击 **"生成 Prompt"** 按钮
- 查看自动生成的 Prompt 模板
- 可以**自由编辑** Prompt 内容
- 点击 **"生成分析"** 按钮

### Step 3: AI 分析
- 系统调用 Gemini API 进行分析
- 等待 10-30 秒（取决于内容长度）
- 显示加载动画

### Step 4: 查看和保存结果
- 在富文本编辑器中查看分析结果
- 可以**编辑和修改**内容
- 点击 **"保存复盘"** 保存到数据库
- 点击 **"导出 Word"** 或 **"导出 PDF"**（开发中）

## 📄 Documents Review 使用流程

### Step 1: 上传文档
- 点击上传区域选择文件
- 支持格式：**PDF、Word (DOC/DOCX)、TXT**
- 显示已选择的文件名

### Step 2: 填写表单
- 输入**字数要求**：500-10,000 字
- 选择**应用场景**：
  - 职场
  - 创业
  - 个人成长
  - 个人财务规划
- 选择**输出语言**：
  - 英文、法文、西班牙文
  - 简体中文、繁体中文
  - 日文

### Step 3: 生成并编辑 Prompt
- 点击 **"生成 Prompt"** 按钮
- 查看自动生成的文档分析 Prompt
- 可以**自由编辑** Prompt 内容
- 点击 **"生成分析"** 按钮

### Step 4: AI 分析
- 系统读取文档内容
- 调用 Gemini API 分析文档
- 等待 10-30 秒
- 显示加载动画

### Step 5: 查看和保存结果
- 在富文本编辑器中查看分析结果
- 可以**编辑和修改**内容
- 点击 **"保存复盘"** 保存到数据库
- 点击 **"导出 Word"** 或 **"导出 PDF"**（开发中）

## 💡 使用技巧

### Prompt 编辑技巧
- **保留结构框架**：【核心主题与核心观点】、【关键概念与框架】等
- **调整重点**：根据需求调整分析重点
- **添加具体问题**：在 Prompt 中加入想了解的具体问题

### 最佳实践
1. **字数要求**：建议 2000-5000 字，既详细又不冗长
2. **应用场景**：选择最匹配的场景，AI 会给出更针对性的建议
3. **输出语言**：选择你最熟悉的语言，便于理解和应用
4. **编辑结果**：AI 生成后可以根据需要修改润色

## 📝 已知限制

### 文档上传限制
- **TXT 文件**：✅ 完全支持
- **PDF 文件**：⚠️ 需要转换为 TXT（暂不支持直接解析）
- **Word 文件**：⚠️ 需要转换为 TXT（暂不支持直接解析）

**建议**：使用在线工具先将 PDF/Word 转换为 TXT 格式

### 导出功能
- **Word 导出**：🚧 开发中（当前为占位符）
- **PDF 导出**：🚧 开发中（当前为占位符）

### API 限制
- **Gemini API**：有请求频率限制
- **内容长度**：单次分析约 30,000 字符
- **超时处理**：如遇超时请稍后重试

## 🔍 故障排查

### 看不到新的 Tabs
1. 检查账号类型（必须是 Admin 或付费会员）
2. 尝试**退出登录重新登录**
3. 清除浏览器缓存并刷新页面

### AI 分析失败
1. 检查网络连接
2. 确认 Prompt 内容不为空
3. 稍后重试（可能是 API 限流）

### 文档上传失败
1. 确认文件格式（PDF、Word、TXT）
2. 文件大小不要太大（建议 < 5MB）
3. 如果是 PDF/Word，先转换为 TXT

### 保存失败
1. 确认已登录
2. 检查网络连接
3. 查看浏览器控制台错误信息
