# 快速测试清单

## 立即执行 (重要！)

### 步骤 1: 清除浏览器缓存 ⚠️

**Windows/Linux**:
```
Ctrl + Shift + R
```

**Mac**:
```
Cmd + Shift + R
```

或者：
1. 按 `F12` 打开开发者工具
2. 右键点击刷新按钮
3. 选择 "清空缓存并硬性重新加载"

### 步骤 2: 测试 Prompt 编辑器

1. 打开任意书籍
2. 找到一个没有内容的小节
3. 点击 "生成内容" 按钮
4. **第1次弹窗**: 输入字数 (如: 1000)
5. **第2次弹窗**: Prompt 编辑器 ← 应该出现！
   - 如果出现：✅ 成功！
   - 如果没出现：❌ 继续下面的步骤

### 步骤 3: 如果 Prompt 编辑器仍不出现

#### 方法 A: 使用无痕模式
```
Chrome: Ctrl + Shift + N
Firefox: Ctrl + Shift + P
```

#### 方法 B: 完全清除缓存
1. 按 `F12`
2. Application 标签
3. Clear site data
4. 点击 "Clear data"

#### 方法 C: 检查控制台错误
1. 按 `F12`
2. Console 标签
3. 查看是否有错误
4. 截图发送给我

### 步骤 4: 验证完整性强调

1. 在 Prompt 编辑器中查看
2. 应该看到 **⚠️【关键】** 标记
3. 应该看到 "内容必须完整，从头到尾一气呵成"
4. 应该看到 "最后一段必须是总结性质的收尾"

### 步骤 5: 测试生成内容

1. 使用默认 Prompt 或编辑后
2. 点击 "确定"
3. 等待生成 (30-60秒)
4. 检查结果：
   - ✅ 字数在 ±10% 范围内
   - ✅ 内容有完整的结尾
   - ✅ 没有 "未完待续" 等字样

## 如果遇到 502 错误

### 方法 1: 减少字数
- 从 2000 字改为 1000 字
- 分多次生成

### 方法 2: 等待重试
- 等待 2-3 分钟
- 重新尝试生成

### 方法 3: 使用编辑功能
- 如果生成失败
- 使用 "编辑" 手动输入

## 验证清单

- [ ] 浏览器缓存已清除 (Ctrl+Shift+R)
- [ ] Prompt 编辑器能正常弹出
- [ ] 看到增强的完整性要求 (⚠️【关键】)
- [ ] 生成的内容字数准确 (±10%)
- [ ] 生成的内容有完整结尾
- [ ] 没有 502 错误 (或重试后成功)
